# **App Name**: Circus Hub

## Core Features:

- Modernized Website: Responsive website with streamlined ticket purchase flows and basic SEO.
- Multi-Vendor Ticketing & Concessions App: Centralized platform for purchasing tickets and ordering concessions from multiple vendors.
- Garden Bros Circus Mobile App: Dedicated app for Garden Bros Circus, focusing on brand engagement and promotions.
- Ticket Validation & QR Scanning: Platform for scanning and validating tickets via QR codes to prevent fraud.
- Real-time Availability: Display real-time ticket availability and seat selection using data stored on Firestore.
- Personalized Recommendations: AI powered tool to generate show recommendations based on viewing history and preferences.
- Push Notifications: Send push notifications for upcoming shows, order updates, and special promotions via Firebase.

## Style Guidelines:

- Primary color: Deep red (#A62929), reflecting the energy and excitement of the circus.
- Background color: Light cream (#F2E8D0), providing a soft contrast to the vibrant primary color.
- Accent color: Bright gold (#D4A94A), used for highlights and call-to-action buttons to evoke a sense of spectacle.
- Body font: 'PT Sans', a humanist sans-serif for readability and a modern feel.
- Headline font: 'Playfair', a modern sans-serif to bring an elegant and fashionable aesthetic to titles.
- Use vibrant and playful icons, relating to circus themes (animals, performers, etc.)
- Incorporate lively animations for transitions and user feedback to enhance the user experience.
